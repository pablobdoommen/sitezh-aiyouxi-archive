# sitezh-aiyouxi-archive

## 项目简介

本仓库用于归档和发布若干独立的 HTML 页面。这些页面内容主要来自对特定主题的收集、整理与再呈现，不隶属于任何特定网站或域名。项目以静态页面形式提供浏览，便于直接查看与分享。

## 目录结构

```
sitezh-aiyouxi-archive/
├── index.html          # 页面导航索引（如有）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、图片、字体等）
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个页面为一个独立单元。
- `assets/` 目录存放页面可能引用的样式、图像等公共资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，无需服务器环境即可在浏览器中直接打开。
- 页面内容可能包含文字、图片、表格等常见网页元素，均为静态呈现。
- 部分页面可能包含简单的交互功能（如按钮、表单演示等），但无后端依赖。
- 页面归档的目的是便于长期保存与离线查看，无需依赖外部服务。

## 维护说明

- 本仓库由项目维护者不定期更新，新增页面或修正已有页面。
- 欢迎通过 Issue 提出建议或报告问题。
- 如需提交新页面，请确保其为独立 HTML 文件，并放置于 `pages/` 目录下。
- 不鼓励在页面中添加外部链接、跟踪代码或商业推广内容。
- 所有页面内容应保持清晰、简洁，符合归档用途。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 下的任意 HTML 文件即可查看。
3. 若存在 `index.html`，可通过其浏览页面列表。
